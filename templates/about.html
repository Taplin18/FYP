What is a Spatial Bloom Filter?

Location privacy is a growing issue in cybersecurity.
The spatial bloom filter is one of the many privacy-preserving protocols aimed at mitigating this threat to personal privacy.
The spatial bloom filter is a data structure based on bloom filters, designed to store location information.
The protocol keeps the user's exact location hidden but allows the provider of the service to learn when the user is close to specific points of interest or inside predefined areas. At the same time, the points and areas of interest remain oblivious to the user.

The Cork dataset

There are four Areas of Interest marked on the map of Cork and each area has a set of coordinates associated with it.
There is a geospatial grid placed on the map which allows the coordinates that are extracted to be accurate to 0.0001 of a degree. The coordinates for are taken from the bottom-left of each cell.
When the dataset is imported, it is the Area od Interest number that is placed in the SBF. Each coordinate is placed in the SBF three times as there is three hash functions.
As there is many values placed into the SBF, the Area of Interest number can be overwritten by the new values that are being inserted.
